# Vaquita

A CHISEL based RISC-V Vector (RVV) v1.0 Extension Coprocessor Generator.

## Dependencies

| Dependencies         | Version         |
| :----------- | :--------------: |
| sbt | 1.10.1 |
| verilator | 5.026 |
